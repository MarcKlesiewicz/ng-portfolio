In the realm of package management, Selvhent stands out as a revolutionary digital warehouse management system tailored specifically for parcel shops. Designed to be simple to operate and install, Selvhent optimizes package storage, scales seamlessly to match the size of any shop, and significantly reduces the time spent on package handling—saving up to two-thirds of the time for both employees and customers.


The system is composed of two primary components:

  1. Customer Unit – A sleek, durable kiosk with an intuitive touchscreen interface that guides customers effortlessly through the process of providing necessary package information.

  2. Warehouse Unit – A tablet-based system that tracks package locations with pinpoint accuracy, identifies unclaimed packages that need to be returned, and revolutionizes how parcel shops organize and retrieve packages.


Notable features:

  • Streamlined package management: Employees gain unparalleled visibility into package inventory, reducing errors and improving efficiency.

  • Ease of use: Customers can easily input delivery couriers, package numbers, and create bulk pickups.

  • Admin management: Manage shops, couriers, and their connections through an intuitive interface.



My role in this project was to implement the web-based admin application using Flutter and to handle pickup requests in the warehouse unit.
